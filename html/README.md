# Villa Booking Website

A responsive Villa Booking Website developed using HTML, CSS, and JavaScript. The application allows users to explore villas, view details, and make booking inquiries through an attractive user interface.

## Features

- Responsive Design
- Home Page
- Villa Listings
- Villa Details
- Booking Section
- Contact Form
- User-Friendly Interface

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

```
Villa-Project/

```

## How to Run

1. Clone the repository:

```bash
git clone <your-repository-link>
```

2. Open the project folder.

3. Run `index.html` in your browser.
